# Food Menu Manager API

## Deskripsi Singkat
- Project menggunakan **Express.js**  
- Mengaplikasikan setup **MVC, routing, CRUD**  
- Data disimpan dalam file JSON  
- API hanya untuk manajemen menu makanan (bukan frontend)
